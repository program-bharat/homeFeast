# HomeFeast

HomeFeast is a full-stack MERN home-cooked food marketplace that connects customers with local home cooks. Users can browse menus, place one-time or subscription meal orders, manage profiles, and track their orders. Cooks can manage menus, orders, earnings, and profiles, while admins oversee platform activity and user management.

## Tech Stack

**Frontend:** React.js, Tailwind CSS, React Router, Axios
**Backend:** Node.js, Express.js
**Database:** MongoDB Atlas
**Authentication:** JWT, bcrypt
**Image Storage:** Cloudinary

## Features

### User
* User registration and login
* JWT-based authentication and authorization
* Browse approved cooks
* View cook profiles and menus
* Add items to cart
* Place one-time and subscription orders
* View order history and order details
* Cancel pending orders
* Update profile information
* Change password

### Cook
* Cook profile management
* Menu creation, update, and deletion
* Order management
* Earnings dashboard
* Overview dashboard
* View customer orders
* Manage availability

### Admin
* Dashboard overview
* User management
* Cook approval and management
* Order monitoring
* Platform analytics
* Earnings overview
